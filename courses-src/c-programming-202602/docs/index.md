## 교과목 개요

| 항목 | 내용 |

## 학습 성과 (CLO)

1. C 기본 문법(변수·연산자·제어문·함수)을 정확히 사용한다.
2. 배열·포인터·구조체로 데이터를 구조화하고 메모리를 이해한다.
3. Arduino 보드에서 센서·액추에이터를 C/C++ 문법으로 제어한다.
4. C 프로그램과 ROS2 노드 간 데이터를 주고받는 시스템을 구현한다.
5. 팀 단위로 모빌리티 응용 SW(로봇 연계)를 설계·구현·발표한다.
