## 학습 목표
- `struct` 정의·멤버 접근, 구조체 배열, 구조체 포인터(`->`)를 사용한다.
- 차량 자세(Pose)/센서 패킷을 구조체로 묶고 직렬화/역직렬화한다.
- (예고) 구조체+포인터로 **연결리스트 한 노드 잇기**를 체험한다(자료구조 가교).
