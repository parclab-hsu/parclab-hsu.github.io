# 📝 정보처리기사(실기) C — 주차별 이론 + 코드 추적 문제

> 정보처리기사 실기 C 파트는 **"이 코드의 실행 결과는?"**(출력 예측)이 핵심이다.
> 아래는 **주차 주제에 맞춰 새로 만든 원본 유형 문제**이며(실제 기출 복제 아님), 모든 코드는 `gcc`로 실행해 정답을 검증했다.
> 각 주제의 **이론**을 먼저 읽고, 문제는 **손으로 추적**해 답을 적은 뒤 접힌 해설을 펼쳐 비교하자. 필기 이론은 [정보처리기사 필기 이론](exam-written.md).

---

## 2주차 · 출력(printf)

**이론** — `%d %f %c %s %x` 자리에 값을 끼운다. 폭·정렬: `%5d`(오른쪽) · `%-5d`(왼쪽) · `%.2f`(소수 2자리). `%%`=기호 `%`.

```c
int a = 5; double b = 3.14159;
printf("[%3d]\n", a);
printf("[%-3d]\n", a);
printf("[%.2f]\n", b);
printf("%d%%\n", 50);
```
??? success "정답 및 해설"
    출력:
    ```
    [  5]
    [5  ]
    [3.14]
    50%
    ```
    - `%3d` 오른쪽 정렬 → `  5`, `%-3d` 왼쪽 정렬 → `5  `, `%.2f` → `3.14`, `%%` → `%`.

## 3주차 · 변수·자료형·형 변환

**이론** — `char`는 작은 정수(ASCII). `'A'`=65. 실수를 정수에 넣으면 **소수 버림**(`(int)3.99==3`).

```c
int  i = 'A';
char c = 66;
double d = 3.99;
int t = (int)d;
printf("%d %c %d", i, c, t);
```
??? success "정답 및 해설"
    **출력: `65 B 3`**
    - `'A'`=**65**, 66→`%c`=**'B'**, `(int)3.99`=**3**(버림).

---

## 4주차 · 연산자

**이론** — 우선순위: 산술(`* / %`) > 시프트 > 관계 > 비트 > 논리 > 대입. 후위 `a++`(쓰고 증가) vs 전위 `++a`(증가하고 씀). 비트 `&|^~ <<`.

```c
int a = 5;
int b = a++;        // 후위
int c = ++a;        // 전위
printf("%d %d %d\n", a, b, c);

int r = 2 + 3 * 4 % 5;
printf("%d\n", r);

int x = 0x0F;       // 15
printf("%d %d %d", x<<2, x&0x09, x^0x05);
```
??? success "정답 및 해설"
    출력:
    ```
    7 5 7
    4
    60 9 10
    ```
    - `b=a++`→b=5,a=6. `c=++a`→a=7,c=7.
    - `3*4=12, 12%5=2, 2+2=`**4**.
    - `15<<2=`**60**, `1111&1001=`**9**, `1111^0101=`**10**.

## 5주차 · 조건문 (switch fall-through)

**이론** — `case` 끝에 `break`가 없으면 **다음 case로 흘러간다(fall-through)**.

```c
int x = 2, s = 0;
switch (x) {
    case 1: s += 1;
    case 2: s += 2;   // break 없음
    case 3: s += 3; break;
    case 4: s += 4;
}
printf("%d", s);
```
??? success "정답 및 해설"
    **출력: `5`**
    - `case 2`부터 `s+=2`(s=2) → break 없어 `case 3`로 흘러 `s+=3`(s=5) → break. → **5**.

---

## 6주차 · 반복문 (break·continue)

**이론** — `continue`(이번 회차 건너뜀) vs `break`(즉시 탈출). `do~while`은 최소 1회.

```c
int sum = 0;
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) continue;   // 짝수 건너뜀
    if (i > 7)      break;      // 7 넘으면 종료
    sum += i;
}
printf("%d", sum);
```
??? success "정답 및 해설"
    **출력: `16`**
    - 홀수만, `i>7`이면 멈춤 → 1+3+5+7 = **16**.

---

## 7·9주차 · 함수·스코프 (static / 값·주소 전달)

**이론** — 값 전달은 복사본(원본 유지), 주소 전달은 원본 수정. `static` 지역변수는 호출 사이 값 유지.

```c
int f(void) { static int c = 0; c += 2; return c; }
void swap(int *x, int *y) { int t = *x; *x = *y; *y = t; }

int main(void) {
    printf("%d ", f()); printf("%d ", f()); printf("%d\n", f());
    int a = 1, b = 2; swap(&a, &b);
    printf("%d %d", a, b);
}
```
??? success "정답 및 해설"
    출력:
    ```
    2 4 6
    2 1
    ```
    - `static c` 유지 → 2,4,6. (한 `printf`에 `f()`를 여러 번 넣으면 평가 순서 미정의 → 따로 호출)
    - 주소 전달 swap → `2 1`. 값 전달이면 `1 2` 그대로(함정).

---

## 10주차 · 배열 (1차원)

**이론** — 연속 공간, 인덱스 0부터, 배열 이름=첫 원소 주소. 삼항 `(조건)?A:B`.

```c
int a[5] = {1, 2, 3, 4, 5};
int s = 0;
for (int i = 0; i < 5; i++)
    s += a[i] * ((i % 2 == 0) ? 1 : -1);
printf("%d", s);
```
??? success "정답 및 해설"
    **출력: `3`**
    - 짝수 인덱스 +, 홀수 − → `+1 −2 +3 −4 +5` = **3**.

---

## 11주차 · 배열(2차원)·문자열

**이론** — `a[i][j]`(i행 j열), `i==j`면 대각선. 문자열은 `'\0'`로 끝나는 char 배열, `while(*p)`=`*p!='\0'`.

```c
int a[2][3] = {{1,2,3},{4,5,6}};
int s = 0;
for (int i = 0; i < 2; i++)
    for (int j = 0; j < 3; j++)
        if (i == j) s += a[i][j];
printf("%d\n", s);

char str[] = "ABCDE";
int n = 0;
for (char *p = str; *p; p++) n++;
printf("%d %c", n, str[n-1]);
```
??? success "정답 및 해설"
    출력:
    ```
    6
    5 E
    ```
    - 대각선 `a[0][0]=1`+`a[1][1]=5` = **6**.
    - 길이 **5**, 마지막 `str[4]`=**'E'**.

---

## 12주차 · 포인터 기초

**이론** — 포인터는 주소를 담는 변수. `&`(주소), `*`(역참조). `*p`로 원본 수정. `p=&b`면 이후 `*p`는 새 대상.

```c
int a = 10, b = 20;
int *p = &a;
*p += 5;      // a 수정
p = &b;       // b 를 가리킴
*p *= 2;      // b 수정
printf("%d %d", a, b);
```
??? success "정답 및 해설"
    **출력: `15 40`**
    - `*p += 5` → a=**15**. `p=&b` 후 `*p *= 2` → b=**40**.

---

## 13주차 · 포인터 활용·재귀

**이론** — `*(p+i)==p[i]`, `p[-1]==*(p-1)`. `*(p+2)`(이동 후 역참조) ≠ `*p+2`(역참조 후 +2). 재귀: `n%10`(끝자리)·`n/10`(제거).

```c
int a[] = {10, 20, 30, 40};
int *p = a;
printf("%d ", *(p+2));
printf("%d ", *p + 2);
printf("%d\n", p[3] - p[0]);

int f(int n) { if (n == 0) return 0; return n % 10 + f(n / 10); }
printf("%d", f(1234));
```
??? success "정답 및 해설"
    출력:
    ```
    30 12 30
    10
    ```
    - `*(p+2)`=**30**, `*p+2`=10+2=**12**, `p[3]-p[0]`=**30**.
    - 자릿수 합 1234 → 4+3+2+1 = **10**.

---

## 14주차 · 구조체

**이론** — 구조체 배열 `arr[i]`, `p=arr`이면 `p->m`=`arr[0].m`, `(p+1)->m`=`arr[1].m`. 값은 `.`, 포인터는 `->`.

```c
struct P { int a, b; };
struct P arr[2] = {{1, 2}, {3, 4}};
struct P *p = arr;
int s = p->a + (p+1)->b;
printf("%d", s);
```
??? success "정답 및 해설"
    **출력: `5`**
    - `p->a`=`arr[0].a`=**1**, `(p+1)->b`=`arr[1].b`=**4** → **5**.

---

## 시험장 체크리스트
- [ ] `printf` 폭/정렬(`%5d`,`%-5d`)·`%.2f`
- [ ] `char`↔정수(ASCII), 실수→정수는 **버림**
- [ ] `*(p+2)` vs `*p+2`, `*(p++)`/`*(++p)`
- [ ] `switch` **fall-through**(break 누락)
- [ ] `continue`(건너뜀) vs `break`(탈출)
- [ ] `static`/전역 값 유지 · 값 vs 주소 전달
- [ ] 배열=주소, `a[i]==*(a+i)`, 대각선 `i==j`
- [ ] 구조체 `->` vs `.`, 비트연산은 2진수로

!!! warning "시험은 스스로"
    시험은 **AI 없이** 손으로 추적하는 힘을 본다. 평소 연습은 AI와, 시험은 스스로.
