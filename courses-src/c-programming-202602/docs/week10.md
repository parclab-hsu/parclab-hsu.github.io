# 10주차 · 배열 (1차원) + LED 막대그래프
> C언어 · 미래모빌리티학과 | CLO2·CLO3 | 교재 Ch10 · 자료 `lectures/11`

![개념도](img/03_array_memory.png)

## 학습 목표
- 배열 선언·초기화·인덱스 접근, 배열과 반복문을 사용한다.
- 배열을 함수에 전달한다(주소 전달의 첫 경험).
- 센서 버퍼·**이동평균 필터**·통계를 구현하고, LED 막대그래프로 시각화한다.

## 강의 흐름 (150분)
| 시간 | 활동 | 내용·자료 |
|------|------|-----------|
| 10 | 도입 | 변수 여러 개의 불편 → 배열 필요 |
| 40 | 개념 | 배열 선언/초기화/순회, 함수 전달 (Ch10) `lectures/11` · 그림 `03_array_memory.png` |
| 25 | 시연 | 이동평균 필터 `ex04_movavg.c` |

## 모빌리티 연계
- 이동평균은 센서 노이즈 제거의 기본기 → LiDAR/거리 데이터 처리로 직결(15주).

## 과제 (10주차)
- 배열 최댓값, 가변 윈도우 이동평균, (도전) 제자리 뒤집기(연습 4-1~4-3).
