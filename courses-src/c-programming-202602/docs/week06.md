## 학습 목표
- `for`, `while`, `do-while`, 중첩 반복, `break`/`continue`를 사용한다.
- 센서를 N회 샘플링하여 합/평균/최대·최소를 구한다.
- **LED 표정 애니메이션**(반복 패턴)을 만든다.
