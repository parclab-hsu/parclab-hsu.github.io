## 학습 목표
- 지역변수·전역변수·`static`의 생존범위와 수명을 구분한다.
- 매개변수 전달 방식(값 전달)을 이해한다.
- 차량 "상태"를 전역/정적 변수로 유지하는 패턴을 작성한다.
