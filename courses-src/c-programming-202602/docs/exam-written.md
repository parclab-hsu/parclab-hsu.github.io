## 1. 프로그래밍 언어 분류

| 패러다임 | 특징 | 예 |
|----------|------|----|
| 절차적(명령형) | 순서대로 실행, 함수 단위 분할 | **C**, 포트란 |

## 2. 데이터 타입·변수·상수

- **기본형**: 정수(`char` 1B, `short` 2B, `int` 4B, `long` 8B), 실수(`float` 4B, `double` 8B), 문자(`char`).
- **부호**: `signed`(기본)/`unsigned`. `unsigned char` 범위 0~255, `signed char` −128~127.
- **상수**: 리터럴(`10`, `3.14`, `'A'`, `"hi"`), `const`(수정 불가 변수), `#define`(전처리 치환), `enum`(열거 상수).
- **변수 명명**: 영문·숫자·`_`만, 숫자로 시작 불가, 대소문자 구분, 예약어 불가.

## 3. 기억 클래스 (storage class) — 필기 단골

변수가 **어디에 저장되고, 얼마나 살아있고, 어디서 보이는가**를 정한다.

| 기억 클래스 | 저장 위치 | 유효 범위(scope) | 생존 기간(lifetime) | 기본 초기값 |
|-------------|-----------|------------------|---------------------|-------------|
| `auto`(기본 지역) | 스택 | 블록 내부 | 블록 종료 시 소멸 | 쓰레기값 |
| `static` | 데이터 영역 | 선언 위치 | **프로그램 종료까지 유지** | 0 |
| `register` | (권고) 레지스터 | 블록 내부 | 블록 종료 시 소멸 | 쓰레기값 |
| `extern` | 데이터 영역 | 다른 파일에서 참조 | 프로그램 종료까지 | 0 |

> 💡 `static` 지역변수는 값이 **호출 사이에 유지**된다(누적 카운터). 전역변수는 여러 파일에서 `extern`으로 공유.

## 4. 연산자 우선순위·결합성

우선순위 높은 → 낮은 순(요약):

| 순위 | 연산자 | 결합성 |
|------|--------|--------|
| 1 | `()` `[]` `->` `.` | 좌→우 |
| 2 | 단항 `!` `~` `++` `--` `-` `*`(역참조) `&`(주소) | 우→좌 |
| 3 | `*` `/` `%` | 좌→우 |
| 4 | `+` `-` | 좌→우 |
| 5 | `<<` `>>` | 좌→우 |
| 6 | `<` `<=` `>` `>=` | 좌→우 |
| 7 | `==` `!=` | 좌→우 |
| 8 | `&` → `^` → `\|` (비트) | 좌→우 |
| 9 | `&&` → `\|\|` (논리) | 좌→우 |
| 10 | `?:` (삼항) | 우→좌 |
| 11 | `=` `+=` `-=` … (대입) | 우→좌 |
| 12 | `,` (콤마) | 좌→우 |

> ⚠ 산술(`* / %`)이 비트·논리보다 **먼저**다. 헷갈리면 괄호. 대입·삼항·단항은 **우→좌** 결합.

## 5. 제어문

- **조건**: `if / else if / else`, `switch~case`(정수·문자, `break` 없으면 fall-through), 삼항 `조건?A:B`.
- **반복**: `for`(횟수), `while`(조건 선검사), `do~while`(**최소 1회** 후검사).
- **분기**: `break`(반복/switch 탈출), `continue`(다음 반복으로), `goto`(레이블 이동, 지양), `return`(함수 종료).

---

## 6. 배열·포인터·문자열

- **배열**: 같은 자료형의 연속 공간, 인덱스 **0부터**. 배열 이름 = 첫 원소 주소(`a == &a[0]`).
- **포인터**: 주소를 담는 변수. `&`(주소), `*`(역참조). `p+1`은 **가리키는 자료형 크기만큼** 이동.
- `a[i] == *(a+i)`. **이중 포인터** `int **pp` 는 포인터의 주소.
- **문자열**: `'\0'`로 끝나는 `char` 배열. `strlen`(길이), `strcpy`(복사), `strcmp`(비교, 0이면 같음), `strcat`(연결).

---

## 7. 구조체·공용체·열거형

| 키워드 | 의미 | 메모리 |
|--------|------|--------|
| `struct` | 여러 자료형을 **묶음**(멤버 각각 공간) | 멤버 크기 합(+ 정렬 패딩) |
| `union` | 멤버가 **같은 공간 공유**(한 번에 하나만) | 가장 큰 멤버 크기 |
| `enum` | 이름 붙은 정수 상수 집합 | 보통 `int` |

- 멤버 접근: 값 변수는 `.`, **포인터는 `->`**(`p->m == (*p).m`).
- `typedef` 로 자료형에 별칭을 준다: `typedef struct{...} Pose;`.

## 8. 함수와 매개변수 전달 방식 — 필기 단골

| 방식 | 넘기는 것 | 원본 변화 | C에서 |
|------|-----------|-----------|-------|
| 값에 의한 호출(call by value) | 값의 **복사본** | ✗ 안 바뀜 | 기본 |
| 주소에 의한 호출(call by reference) | 변수의 **주소** | ✓ 바뀜 | 포인터로 구현 |

```c
void inc(int *p) { (*p)++; }   // 주소를 받아 원본 증가
int a = 5; inc(&a);            // a == 6
```
- **재귀(순환) 함수**: 자기 자신 호출. **종료 조건** 필수(없으면 스택 오버플로).
- 함수는 `return` 으로 값 **하나**만 반환 → 여러 결과는 **포인터 매개변수**로.

---

## 9. 진법과 비트 (자주 출제)

- 진법 접두: `0x`(16진), `0`(8진), 접두 없음(10진). 예: `0x2A == 42`, `052 == 42`.
- **보수**: 1의 보수=비트 반전, **2의 보수=1의 보수+1**(음수 표현). `signed char -5` 의 비트는 `1111 1011`(= unsigned로 251).
- 비트 연산: `&`(AND) `|`(OR) `^`(XOR) `~`(NOT) `<<`(왼쪽 시프트=×2ⁿ) `>>`(오른쪽=÷2ⁿ).

```
5 & 3 = 1     5 | 3 = 7     5 ^ 3 = 6     ~5 = -6     1 << 4 = 16
```

## 10. 라이브러리·전처리기

- **표준 헤더**: `<stdio.h>`(입출력), `<stdlib.h>`(변환·난수·메모리), `<string.h>`(문자열), `<math.h>`(수학, 컴파일 `-lm`).
- **전처리기(#)**: `#include`(헤더 삽입), `#define`(매크로 치환), `#ifdef/#endif`(조건부 컴파일).
- 컴파일 과정: **전처리 → 컴파일 → 어셈블 → 링크** → 실행파일.

## 확인문제 (OX)

각 문장이 맞으면 O, 틀리면 X. **먼저 스스로 판단**한 뒤 정답을 펼치자.

1. `static` 지역변수는 함수 호출이 끝나면 값이 사라진다.
2. 값에 의한 호출(call by value)로 넘기면 함수 안에서 원본 변수를 바꿀 수 있다.
3. `do~while` 문은 조건이 처음부터 거짓이면 본문을 한 번도 실행하지 않는다.
4. 배열 이름은 그 배열 첫 번째 원소의 주소를 의미한다.
5. `union`은 모든 멤버가 각각 독립된 메모리 공간을 가진다.
6. C에서 `int / int` 나눗셈은 소수점 이하를 버린다.
7. 산술 연산자(`*`,`/`)는 관계 연산자(`<`,`>`)보다 먼저 계산된다.
8. 2의 보수는 1의 보수에 1을 더한 값이다.
9. 구조체 **포인터**에서 멤버에 접근할 때는 `.` 연산자를 쓴다.
10. `register`로 선언하면 변수는 반드시 CPU 레지스터에 저장된다.
11. `strcmp(a, b)` 가 0을 반환하면 두 문자열이 서로 다르다는 뜻이다.
12. 전처리기 `#define` 매크로는 프로그램 실행 중에 함수처럼 호출된다.

??? success "정답 및 해설"
    1. **X** — `static` 지역변수는 호출이 끝나도 값이 **유지**된다(데이터 영역).
    2. **X** — 복사본이 넘어가므로 **원본은 안 바뀐다**. 바꾸려면 주소(포인터) 전달.
    3. **X** — `do~while`은 **본문을 먼저 실행**하므로 **최소 1회** 실행된다.
    4. **O** — `a == &a[0]`. 그래서 `a[i] == *(a+i)`.
    5. **X** — `union`은 멤버들이 **같은 공간을 공유**(가장 큰 멤버 크기)한다.
    6. **O** — 정수끼리 나누면 소수를 버린다(`7/2==3`). 실수는 `(double)a/b`.
    7. **O** — 우선순위: 산술 > 시프트 > 관계 > 비트 > 논리 > 대입.
    8. **O** — 2의 보수 = (비트 반전=1의 보수) + 1. 음수 표현에 쓴다.
    9. **X** — 포인터는 `->`(화살표). `.`은 구조체 **변수**용. `p->m == (*p).m`.
    10. **X** — `register`는 **권고**일 뿐, 실제 저장 위치는 컴파일러가 정한다.
    11. **X** — `strcmp`가 0이면 두 문자열이 **같다**는 뜻이다.
    12. **X** — `#define`은 **컴파일 전(전처리) 글자 치환**이다. 실행 중 호출이 아니다.
