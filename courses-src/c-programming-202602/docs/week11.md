## 학습 목표
- 2차원 배열과 문자 배열(문자열 기초)을 사용한다.
- 8×12 점유격자(occupancy grid) 개념을 2차원 배열로 표현한다.
- **WiFi로 보드를 제어**하고, 통신이 ROS2의 전 단계임을 이해한다.
