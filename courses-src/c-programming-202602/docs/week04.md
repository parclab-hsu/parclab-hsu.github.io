## 학습 목표
- 산술·관계·논리·대입·증감 연산자와 우선순위를 사용한다.
- **비트 연산자**(`& | ^ ~ << >>`)로 센서 상태 플래그를 다룬다.
- 속도=거리/시간 등 모빌리티 수식을 구현한다.
