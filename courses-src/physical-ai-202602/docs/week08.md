# 8주차 — 중간고사 (이론 + 실습 평가)

!!! abstract "학습목표"
    1~7주차 내용(ROS2·TF·센서·시뮬레이션·제어·SLAM)을 종합 평가한다.

## ⏱️ 3시간 구성

| 교시 | 시간 | 내용 |
| --- | --- | --- |
| 1교시 | 50분 | 이론 평가 (서술/객관식) |

## 📝 1교시. 이론 평가 (예시 문항)

1. Digital AI와 Physical AI의 차이를 Sense–Plan–Act 관점에서 설명하라.
2. ROS 2의 토픽·서비스·액션의 차이와 각각의 적절한 사용 예를 들라.
3. TF 트리에서 world→odom→base_link→sensor 구조가 필요한 이유는?
4. 차동구동 로봇의 cmd_vel(v, ω)이 바퀴 속도로 어떻게 변환되는가?
5. SLAM과 순수 오도메트리(odometry only)의 차이와 drift 문제를 설명하라.

## 🛠️ 2교시. 실기 — 라이브 코딩 (예시 과제)

!!! note
    제한시간 내 다음 노드를 작성·실행하고 결과를 제출.

**과제 A** — `/scan`을 구독하여 좌·우·전방 거리를 동시에 출력하는 노드 작성

```bash
ros2 launch turtlebot3_gazebo turtlebot3_world.launch.py
# 이후 작성한 scan_monitor 노드 실행
```

**과제 B** — cmd_vel로 특정 거리 전진 후 정지하는 노드 작성(/odom 활용)

## 🛠️ 3교시. 실기 — 통합 과제

**과제 C** — 장애물이 전방 0.4m 이내에 있으면 멈추고, 아니면 전진하는 단순 장애물 회피 노드 완성

- 평가 포인트: `/scan` 구독 · 조건 분기 · cmd_vel 발행 · 동작 시연
