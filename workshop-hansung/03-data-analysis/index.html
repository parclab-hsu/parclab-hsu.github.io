    데이터 분석¶

    이 장에서는 깔끔하게 정리된 예제 대신, 실제 업무에서 만나는 행정 엑셀 파일의 불편함을 그대로 다룹니다. 데모·실습·검증의 짧은 루프를 반복하면서 AI 에이전트를 활용한 데이터 분석 흐름을 익힙니다.

    📥 실습 데이터 — 이 장에서 쓰는 엑셀 4종 × 3개년(총 12개)입니다. 전체 압축 내려받기 (ZIP) — 압축을 풀면 data 폴더가 나옵니다.

    오늘의 데이터 지도¶

    대학 행정 데이터 4종을 3개년으로 비교합니다.

    주제 파일 수 읽어낸 행 수 분석 질문
    전공과목 성적 분포 3 3,177 성적 분포가 어떻게 바뀌었나?
    입학전형 유형별 선발 결과 3 80 전형별 등록 규모와 등록률은?
    외국학생 현황 3 79 외국학생 수와 국가 구성은?
    외국학생 중도탈락 현황 3 78 중도탈락률은 어느 해에 높았나?
    • 분석은 정답 찾기보다 질문을 세우고 확인하는 과정으로 진행합니다.
    • 각 주제마다 합계 행, 기준연도, 단위, 빈칸 반복 여부를 먼저 확인합니다.

    좋은 실습 자료인 이유¶

    이 파일들은 처음부터 정리된 데이터가 아니라 실제 업무에서 만나는 문제를 그대로 담고 있습니다.

    • 문제 1: 겉보기 오류 — 엑셀 내부의 표 범위 정보가 A1로 저장되어 일반 판독기는 한 칸짜리 파일처럼 착각할 수 있습니다.
    • 문제 2: 병합 헤더 — 제목 행, 설명 행, 2단 헤더, 합계 행이 섞여 있어 바로 표 분석을 하기 어렵습니다.
    • 문제 3: 빈칸 반복 — 학과명, 전형유형, 기준연도 등이 첫 행에만 있고 아래 행은 빈칸이라 위 행 값을 이어받아야 합니다.

    읽기 실패를 좋은 질문으로 바꾸기¶

    한 칸만 읽었다면 '다시 해줘'가 아니라 원인을 묻습니다.

    실패했을 때 이어가는 프롬프트 예시는 다음과 같습니다.

    엑셀 파일을 읽었을 때 한 칸만 보이거나 행 수가 이상하면,
파일 내부 구조와 실제 셀 태그 수를 확인해서 원인을 설명해줘.
가능하면 데이터를 복원하는 방법도 제안해줘.
    • 나쁜 반응 — "엑셀이 이상한가 봐요 / AI가 못 읽네요 / 다른 파일로 넘어갈게요"처럼 포기하는 방식은 개선의 기회를 잃습니다.
    • 좋은 반응 — "왜 한 칸으로 보였는지 설명해줘 / 실제 셀 수와 표 범위를 비교해줘 / 복원 가능한 읽기 방식을 제안해줘"처럼 원인을 묻습니다.
    • 참가자 역할 — 이 장면에서 참가자는 사용자라기보다 검증자입니다. AI가 말한 근거를 원본 파일과 맞춰 봅니다.

    AI 에이전트 분석 흐름¶

    한 번에 끝내지 않고 짧은 루프를 반복합니다.

    단계 이름 내용
    1 탐색 파일/시트/행·열
    2 정제 헤더/빈칸/단위
    3 분석 추이/비교/순위
    4 검증 합계/샘플/해석
    5 정리 차트/슬라이드
    • 핵심 포인트 — AI 에이전트에게 '분석해줘'라고만 말하면 결과는 그럴듯하지만 약합니다. 먼저 자료를 확인시키고, 이상한 점을 찾게 한 뒤 분석을 요청해야 합니다.
    • 반복 질문 예시 — "방금 계산한 합계가 원본의 합계 행과 맞아? / 빈칸은 위 행 값으로 채워야 하나? / 파일명 연도와 기준연도가 같은 뜻이야?"

    실습은 세 번 끊어서 진행합니다¶

    한 번에 완성하지 말고, 중간 결과를 확인할 수 있는 지점에서 멈춥니다.

    • 멈춤 1: 파일 목록 — 파일 수, 주제, 연도, 시트 구조가 맞는지 확인합니다.
    • 멈춤 2: 정제 규칙 — 어떤 행을 제외하고, 어떤 빈칸을 채우고, 어떤 단위를 쓸지 정합니다.
    • 멈춤 3: 분석 결과 — 차트와 표를 보기 전에 합계, 기준연도, 샘플 행을 확인합니다.

    각 멈춤 지점에서 '지금 무엇을 확인했나요?'를 스스로 점검합니다. 결과가 다르면 틀린 것을 찾는 것이 아니라 프롬프트와 파일 경로 차이를 봅니다. 화면마다 결과가 달라도 검증 질문은 같습니다.

    실습 1: 데이터 목록과 구조 파악¶

    직접 입력해 볼 수 있는 첫 번째 분석 프롬프트입니다.

    실습 프롬프트 예시는 다음과 같습니다.

    data 폴더의 모든 xlsx 파일을 열어보고,
1) 파일명, 2) 주제, 3) 파일명 연도, 4) 실제 데이터 행 수,
5) 헤더가 시작되는 행, 6) 분석 시 주의점을 표로 정리해줘.

결과를 내기 전에 파일을 실제로 읽었는지 근거를 보여줘.
    • 함께 확인할 것 — 파일이 12개인지 / 4개 주제 × 3개년 구조인지 / 엑셀 표 범위 문제를 발견했는지 / 행 수가 제목 한 줄로만 나오지 않는지
    • 결과가 이상할 때 — AI에게 "일반 판독기로 읽은 행 수와 내부 XML의 셀 수를 비교해줘"라고 지시합니다. 이 장면이 에이전트 활용법의 핵심입니다.

    실습 1의 완료 기준¶

    프롬프트를 실행했다는 것과 자료를 이해했다는 것은 다릅니다.

    확인 항목 좋은 결과 다시 물어볼 신호
    파일 수 12개 파일을 주제별·연도별로 구분 일부 파일만 읽거나 파일명을 추측
    주제 구분 성적, 입학전형, 외국학생, 중도탈락을 분리 파일명 일부만 보고 같은 주제로 묶음
    행 수 제목 행과 실제 데이터 행을 구분 모든 파일을 1행 또는 0행으로 보고
    주의점 병합 헤더, 빈칸 반복, 기준연도 차이를 언급 바로 차트부터 만들려고 함

    실습 2: 정제 기준 정하기¶

    병합 헤더와 빈칸 반복을 분석 가능한 표로 바꿉니다.

    실습 프롬프트 예시는 다음과 같습니다.

    전공과목 성적 분포 파일을 분석 가능한 형태로 정리하려면
어떤 열은 위 행 값을 채워야 하고,
어떤 행은 제외해야 하는지 규칙을 제안해줘.

그 규칙으로 2023-2025년 성적 분포를 A/B/C/D/F 구간으로 요약해줘.
    • 정제 규칙 예시 — 제목/작성자 행 제외 / 헤더 행 이후만 사용 / 학과명·구분·학기 등은 빈칸일 때 위 행 값으로 채움 / 등급 표기는 공백 제거: A + → A+
    • 검증 규칙 예시 — 등급별 학생수 합계가 성적인정 학생총수와 크게 어긋나지 않는지 확인 / 연도별 합계와 차트 값이 같은지 확인

    정제 결과는 규칙으로 설명돼야 합니다¶

    숫자가 나왔다는 사실보다, 어떤 행과 열을 어떻게 처리했는지가 중요합니다.

    • 제외 — 제목 행, 작성자 행, 빈 설명 행, 합계 중복 행을 어떻게 제외했는지 표시합니다.
    • 채우기 — 학과명, 전형유형, 기준연도처럼 위 행 값을 이어받은 열을 적습니다.
    • 단위 — 명, %, 건수처럼 차트에 들어갈 단위를 분리합니다.
    • 근거 — 원본 행 샘플과 정제 후 행 샘플을 3개 이상 비교합니다.

    정제 규칙이 설명되지 않으면 차트가 맞아 보여도 신뢰하기 어렵습니다. "AI가 어떤 행을 버렸는지 알고 있나요?"를 스스로에게 물어보면 검증 감각이 생깁니다.

    외국학생 총원은 3년간 증가¶

    2023년 380명에서 2025년 506명으로 증가했습니다.

    연도 외국학생 총원
    2023 380
    2024 400
    2025 506

    외국학생 총원은 2023→2025년에 126명 증가했습니다. 이 변화는 이후 국가별 구성 변화와 함께 읽어야 합니다.

    이 차트만 보고 결론을 낸다면 무엇을 놓칠 수 있을까요?

    📊 인터랙티브 차트는 발표 슬라이드에서 볼 수 있습니다.

    차트에서 발표 문장 만들기¶

    숫자 하나를 결론, 근거, 주의점으로 나눠 말합니다.

    • 결론 — 외국학생 총원은 2023년 380명에서 2025년 506명으로 증가했습니다.
    • 근거 — 3년 사이 126명 증가했고, 2024년보다 2025년 증가 폭이 큽니다.
    • 주의점 — 총원 증가만으로 지원 수요를 단정하지 말고 국가별 구성과 중도탈락률을 함께 봐야 합니다.

    실습 질문 | 여러분의 차트에서도 결론, 근거, 주의점을 각각 한 문장으로 분리해 보세요.

    2025년에는 베트남 학생 비중이 큽니다¶

    상위 국가 구성이 전체 증가 해석의 핵심입니다.

    국가 2025년 외국학생 수
    베트남 333
    중국 68
    일본 43
    미얀마 28
    몽골 15

    2025년 상위 국가 중 베트남이 333명으로 가장 많습니다. 전체 증가가 특정 국가의 변화로 얼마나 설명되는지 물어볼 수 있습니다.

    학생지원, 수업 운영, 언어 지원 관점에서 어떤 후속 질문이 필요할까요?

    📊 인터랙티브 차트는 발표 슬라이드에서 볼 수 있습니다.

    해석은 다음 질문으로 끝냅니다¶

    데이터가 답한 것과 아직 답하지 못한 것을 구분합니다.

    • 데이터가 말한 것 — 2025년 외국학생 상위 국가 중 베트남 학생 수가 가장 큽니다. 국가별 구성은 수업 운영과 학생 지원 논의의 출발점이 됩니다.
    • 데이터만으로 말할 수 없는 것 — 왜 특정 국가 학생이 늘었는지, 학업 적응도가 어떤지, 지원 요구가 무엇인지는 추가 자료가 필요합니다.

    좋은 인사이트는 단정으로 끝나지 않고 다음 확인 질문을 남깁니다. AI에게 "이 데이터로 말할 수 없는 것은 무엇인가?"를 꼭 물어보세요.

    중도탈락률은 기준연도부터 확인¶

    파일명 연도와 실제 기준연도는 한 해 차이입니다.

    공시 연도 기준연도 등록인원 탈락인원 탈락률
    공시 2023 2022 364 16 4.4%
    공시 2024 2023 380 24 6.3%
    공시 2025 2024 400 13 3.2%

    공시 2024 파일의 기준연도는 2023년이며, 이 해의 중도탈락률 6.3%가 3년 중 가장 높습니다. 파일명 연도와 기준연도를 혼동하면 해석이 달라집니다.

    혼동하면 생기는 오해는 무엇일까요?

    📊 인터랙티브 차트는 발표 슬라이드에서 볼 수 있습니다.

    전형별 등록인원 구성이 달라졌습니다¶

    수능위주, 학생부교과, 학생부종합의 규모 변화를 비교합니다.

    전형 2023 2024 2025
    수능위주 394 535 535
    학생부교과 533 472 490
    학생부종합 539 496 368
    실기위주 152 152 144
    기타 35 36 8

    2024–2025년에는 수능위주 등록인원이 높게 나타나고, 학생부종합은 2025년에 줄어듭니다. 모집 구조 변화와 함께 해석해야 합니다.

    전형별 등록인원 변화와 수업 운영은 어떤 방식으로 연결될 수 있을까요?

    📊 인터랙티브 차트는 발표 슬라이드에서 볼 수 있습니다.

    성적 분포는 A/B 구간 변화가 핵심입니다¶

    2023년 A 비율이 높고, 2024–2025년에는 B 비율이 커집니다.

    연도 A (%) B (%) C (%) D (%) F (%)
    2023 49.6 24.1 20.3 3.1 2.9
    2024 40.9 34.8 19.1 2.6 2.6
    2025 43.2 33.9 18.3 2.5 2.1

    A구간 비율은 2023년 49.6%, 2024년 40.9%, 2025년 43.2%입니다. 평가정책, 과목구성, 수강생 규모 차이를 함께 확인해야 합니다.

    성적 분포 변화에서 데이터만으로 말할 수 있는 것과 말할 수 없는 것은 무엇일까요?

    📊 인터랙티브 차트는 발표 슬라이드에서 볼 수 있습니다.

    분석 결과를 말하기 전에 선을 긋습니다¶

    집계 데이터의 변화와 원인 설명은 다른 층위입니다.

    • 말할 수 있음 — 연도별 A/B/C/D/F 비율이 어떻게 달라졌는지 설명할 수 있습니다.
    • 추가 확인 필요 — 평가 정책, 과목 구성, 수강생 규모 변화가 영향을 줬는지 확인해야 합니다.
    • 말하면 안 됨 — 개별 교수의 평가 방식이나 학생 수준을 집계 자료만으로 단정하지 않습니다.

    해석 경계 확인 프롬프트 예시는 다음과 같습니다.

    방금 분석 결과에서
1) 데이터로 직접 말할 수 있는 것,
2) 추가 자료가 있어야 말할 수 있는 것,
3) 단정하면 안 되는 것을 구분해줘.

    참가자 실습: 인사이트 1개 만들기¶

    정답보다 근거가 있는 한 문장을 만드는 훈련입니다.

    단계 내용 예시
    1. 질문 질문 하나 선택 어느 국가가 이끌었나?
    2. 분석 파일과 계산 기준 지정 국가별 총원 증감
    3. 검증 합계·상위 샘플 확인 증가 합계 비교

    실습 프롬프트 예시는 다음과 같습니다.

    외국학생 현황 2023-2025 파일에서
국가별 총원 증감을 계산하고,
증가분이 큰 상위 5개 국가를 표로 보여줘.
합계 증가분과 국가별 증가분 합계가 맞는지도 확인해줘.

    공유할 때는 한 문장만 남깁니다¶

    결과 공유는 길게 발표하지 않고 구조를 맞춥니다.

    항목 내용
    질문 궁금한 질문
    근거 확인한 파일과 숫자
    검증 다시 확인한 기준
    다음 추가 확인 대상

    공유 구조 예시는 다음과 같습니다.

    • 공유 구조 — 질문: ○○ / 근거: △△ 파일 / □□ 숫자 / 검증: 합계·기준연도 / 다음: ◇◇ 추가 확인
    • 말할 때 기준 — 30초 안에 끝내기 / 숫자는 하나만 강조 / 모르는 부분은 다음 질문으로 남기기
