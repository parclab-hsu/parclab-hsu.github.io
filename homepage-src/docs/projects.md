# 프로젝트 (Research Projects)

PARC Lab에서 진행 중인 정부과제, 산학협력, 자체 연구 프로젝트입니다. 각 프로젝트는 Physical AI와 실시간 제어의 실용적 적용을 목표로 합니다.

---

## :material-progress-clock: 진행 중인 프로젝트

<div class="grid cards" markdown>

-   :material-circle:{ style="color:#2e9e44" } __진행중 · 달 탐사 로봇 디지털 트윈 및 HILS 플랫폼__

    ---

    Isaac Sim + OmniLRS 기반 달 탐사 로버 디지털 트윈 플랫폼 개발. ROS2 통합, 강화학습(RL) 기반 자율주행 정책 학습, HILS 검증 환경 구축. KASA 연계 탐사 임무 MBSE/Digital Twin 연구 수행.

    `Isaac Sim` · `OmniLRS` · `ROS2` · `HILS` · `Reinforcement Learning` · `SysML` · `Jetson Orin`

-   :material-circle:{ style="color:#2e9e44" } __진행중 · EtherCAT 기반 실시간 모션 제어 시스템__

    ---

    EtherCAT 마스터 기반 고속 실시간 제어 시스템 개발. 산업용 서보 드라이브 다축 동기화, DS402 프로파일 구현, 리눅스 RT 환경 최적화. 모빌리티 플랫폼 및 협동 로봇 적용.

    `EtherCAT` · `SOEM` · `Linux RT` · `DS402` · `C/C++` · `ROS2`

-   :material-circle:{ style="color:#2e9e44" } __진행중 · 스마트 모빌리티 보조 시스템 (HSU Smart Cart)__

    ---

    이동약자를 위한 AI 기반 보행 보조 및 자율 모빌리티 시스템. 300kg 듀얼 500W 모터 플랫폼에 완전 어드미턴스 제어 구현. 로드셀 입력, 차동 구동 운동학, PySerial Arduino 통합.

    `Admittance Control` · `Python` · `Arduino` · `Differential Drive` · `Load Cell` · `PySerial`

-   :material-circle:{ style="color:#2e9e44" } __진행중 · 착용형 보행 보조 로봇 임상 효과 검증__

    ---

    성북 시니어 복지관 노인 대상 착용형 보행 보조 로봇 임상 적용 연구. 10MWT 보행 능력 평가, 심리·사회적 수용도 조사, 정부 성과지표 보고서 작성.

    `Wearable Robot` · `Clinical Study` · `10MWT` · `Data Analysis`

</div>

## :material-lightbulb-on-outline: 계획 중인 프로젝트

<div class="grid cards" markdown>

-   :material-circle-outline:{ style="color:#e0a800" } __계획중 · Lean VLA 제조·물류 로봇 적용 연구__

    ---

    경량화 Vision-Language-Action 모델을 제조·물류 로봇에 적용. LoRA 기반 태스크 전환, Flow Matching Policy Head(π0), Grounding DINO + SAM + FoundationPose 인식 스택 통합.

    `VLA` · `LoRA` · `Diffusion Policy` · `Grounding DINO` · `SAM`

-   :material-circle-outline:{ style="color:#e0a800" } __계획중 · 수술 로봇 정렬 유도 시스템 (Align-Metrics)__

    ---

    6축 IMU 기반 수술 중 정렬 유도 시스템 개발. HJC 추정(SCoRE/LM), LSTM 기반 자기장 없는 관절 각도 추정, 무릎 치환술 적용.

    `IMU` · `LSTM` · `SCoRE` · `Surgical Robot` · `HJC Estimation`

-   :material-circle-outline:{ style="color:#e0a800" } __계획중 · Physical AI 기반 협동 로봇 품질 검사__

    ---

    협동 로봇 + F/T 센서 + Vision을 활용한 가전제품 감촉 품질 자동 평가 시스템. 햅틱 피드백 파라미터 분석(피크 힘, 히스테리시스, DTW, CV%) 및 ML 분류.

    `Collaborative Robot` · `F/T Sensor` · `ArUco` · `ML` · `Quality Inspection`

-   :material-circle-outline:{ style="color:#e0a800" } __계획중 · AI 기반 대학 교육 인프라 구축__

    ---

    AI-LMS, HSU AI-Hub, AI Scholar(소크라테스식 에이전트) 등 대학 AI 기본교육 인프라 개발. 데이터 거버넌스(Data Owner, RBAC) 프레임워크 적용.

    `AI Education` · `LMS` · `LLM Agent` · `RBAC` · `Data Governance`
